# Contributing to PlaywrightAIExtractor

We welcome contributions from the community! Please read these guidelines before contributing.

## Getting Started

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and ensure all tests pass.

## Code Style

Follow the existing code style and formatting. We use ESLint and Prettier for code formatting.

## Testing

Write tests for your code using Jest or Mocha. Ensure that all tests pass before submitting a PR.

## Submitting a Pull Request

1. Ensure your commit messages are clear and concise.
2. Submit a pull request and describe the changes you've made.

## Code of Conduct

Please adhere to our [Code of Conduct](CODE_OF_CONDUCT.md) when interacting with the community.
